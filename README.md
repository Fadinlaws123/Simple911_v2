# Simple911 v2

A modern, standalone-first emergency call and dispatch resource for FiveM.

## Planned Features

- Configurable 911, 311, and custom call services
- Templated emergency messages
- Modern caller form
- Live responder dispatch panel
- Claim, respond, waypoint, and resolve actions
- Server-authoritative call state
- Configurable ACE responder permissions
- Call blips and notifications
- Discord logging
- Clean exports and events for integrations
- Optional QBCore, ESX, and ND_Core bridges

## Development Status

Simple911 v2 is currently under active development. The standalone core will be built and tested first before optional framework integrations are added.

> This resource is not yet considered production-ready.
